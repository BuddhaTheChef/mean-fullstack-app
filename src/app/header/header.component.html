Downloads and Installations

Downloading Node

1. Check to see what node version you have by typing "node --version" if nothing is found move to step 2.

2. Go to Node.js Download

3. Install latest stable version and click download

4. Follow installation procedures

5. Check to see if you Dowloaded it by typing "node --version"

Downloading Express

1. Must have node installed before going further

2. Go to a project folder of your terminal and type "npm install express --save"

3. Check to see if you Dowloaded it by creating a new js page and type "var express = require('express') var app = express()"

Downloading Angular

1. Must have node installed before going further

2. Go to the root of your terminal and type "npm install -g @angular/cli"

3. At the end of the install process a prompt will come up asking you to share anonymous useage data with the Angular team type "y" for Yes or "n" for No

to chek if the download was successful type "ng version"

Downloading MongoDB
